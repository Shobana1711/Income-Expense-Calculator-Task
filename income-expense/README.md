# Income Expense Calculator

## Description
A simple web application to track income and expenses with CRUD functionality and local storage support.

## Features
- Add income and expense
- Edit and delete entries
- Filter (All / Income / Expense)
- Local storage support
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
  
## How to Use:
- Enter description and amount
- Select type (income/expense)
- Click Add
- Use Edit/Delete as needed
- Filter using radio buttons
